# DecoreHome - Proyecto Web

Proyecto web de e-commerce para gestión de productos, usuarios y carrito de compras, desarrollado con HTML, CSS y JavaScript.

DecoreHome es un sitio web que simula un sistema de ventas en línea de productos de decoración.
Permite a los usuarios registrarse, iniciar sesión, agregar productos al carrito, gestionar pedidos y visualizar información de contacto.
Incluye también una interfaz de administrador para gestionar usuarios y productos.

HTML5
CSS3 (con diseño responsivo)
JavaScript (manipulación del DOM y almacenamiento local con localStorage)
Font Awesome (íconos)
